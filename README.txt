 Проект 5 (Юнит 4: Основные алгоритмы машинного обучения. Часть I)

 Название проекта: "Компьютер говорит «Нет»"

 Цель проекта: С помощью данных о клиентах банка обучить логистическую регрессию, которая будет предсказывать шанс дефолта со стороны
 возможного клиента

 Задачи проекта:
 - провести разведывательный анализ данных, корреляционный анализ;
 - подготовить данные для обучения логистической регрессии;
 - подобрать наилучшие параметры регрессии для достижения хороших метрик;

 Выводы:
 Был проведен разведывательный анализ данных, корреляционный анализ. Признаки были разделены на группы - бинарные, непрерывные, катего-
 риальные, и для каждой группы была оценена важность.

 Были обучены 3 модели логистической регрессии: по всему датафрейму, используюя undersampling, с подбором гиперпараметров. Финальная
 версия была использована для получения предсказания. Значение полученной метрики ROC AUC - 0.73567

 Дополнтельно: 
 Нужно отметить, что в очередной раз графики Plotly вызывают некоторые сложности. Несмотря на свою чрезвычайную наглядность и интерак-
 тивность, они требуют очень большого количества ресурсов, и при скачивании ноутбука с Kaggle и открытии в Jupyter Notebook не ото-
 бражаются. Видимо необходимо более внимательно приглядеться к библиотеке Seaborn, куда менее требовательной.
